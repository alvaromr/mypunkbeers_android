# My Punk Beers App

## About this project

This is a simple project to play a bit with some new concepts on Android.

It uses [Punk's API](https://punkapi.com/documentation/v2) to fetch and display beers.

## Main Features

+ Coroutines & Flow
+ MVI Clean Architecture
+ Single Activity
+ Hilt
+ Ktor
+ Jetpack Compose
+ Jetpack Navigation
+ Coil
+ SqlDelight
+ Flipper
